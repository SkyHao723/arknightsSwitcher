# 明日方舟切换器 (Arknights Switcher)

- 一个用于在明日方舟官服和B服之间快速切换的工具。
- 目前适配游戏版本：V70.0.0
- 对应服务器文件下载：https://wwaua.lanzouq.com/ipFfE3ibpdyf

## 功能特点

- 🚀 一键切换游戏版本（官服/B服）
- 🔧 自动复制对应版本文件到游戏目录
- 🎯 智能检测并启动游戏
- ⚙️ 配置文件管理游戏路径
- 💻 简洁的命令行界面

## 文件说明

### 主要程序文件
- `Launcher.exe` - 主启动器程序
- `launch-with-C.exe` - 官服启动程序  
- `launch-with-B.exe` - B服启动程序
-（下面两个文件可以单独使用，前提是需要有config.json）

### 配置文件
- `config.json` - 游戏目录配置文件
- `banner.txt` - 启动时显示的欢迎横幅

### 文件结构
```
├── Launcher.exe           # 主程序
├── launch-with-C.exe      # 官服启动程序  
├── launch-with-B.exe      # B服启动程序
├── C2B/                   # B服专用文件目录
│   └── [B服相关文件]
└── B2C/                   # 官服专用文件目录  
    └── [官服相关文件]
```

## 使用方法

### 首次使用
1. 运行 `Launcher.exe`
2. 程序会提示输入明日方舟游戏安装目录
3. 输入完整路径后按回车确认

## 注意事项

⚠️ **重要提醒**：
- 请确保以管理员权限运行程序
- 游戏目录路径不能包含中文字符
- 备份重要游戏数据以防意外
- 不同版本间切换时会覆盖部分游戏文件

## 故障排除

常见问题及解决方案：

**Q: 程序无法找到游戏文件**
A: 检查 `config.json` 中的目录路径是否正确

**Q: 启动失败提示权限不足**
A: 以管理员身份运行程序

**Q: 复制文件失败**
A: 确保目标目录有写入权限，关闭正在运行的游戏进程
